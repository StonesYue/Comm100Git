(warning: all of the command need to run in the root of comm100_UI_Automation folder)
1, install node.js(install the node-v7.7.1-x64.msi under the software folder)
2, install java, config the environment variable(for selenium,if use WebDriver stand alone: no need)
2, install nightwatch(command: npm install)
3, install nightwatch-html-reporter(command: npm install nightwatch-html-reporter) 
4, run case(command: node nightwatch --test tests\comm100Demo.js)
5, create your case and run(we need put our cases under tests folder)
